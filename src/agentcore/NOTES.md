## Compatibility

| OS              | Supported   | Notes                          |
| --------------- | ----------- | ------------------------------ |
| Debian / Ubuntu | ✓ tested    | Primary target                 |
| Alpine Linux    | ✓ supported | Not tested in CI               |

**Architectures:** x86_64, aarch64

## Requirements

Node.js 20 or later is required. If not present, the installer bootstraps Node.js automatically via `nodesource` (Debian/Ubuntu) or `apk` (Alpine).
